--------------------------------------------------------------------------------
CENSUS PROTOCOL
--------------------------------------------------------------------------------
Inclusion Criteria: All woody stems ≥10 cm DBH
DBH Measurement: Diameter tape at 1.3 m height
Height Estimation: Visual estimation (no clinometer available)
Tagging: Unique alphanumeric ID assigned to each tree
Spatial Coordinates: Local plot coordinates (x, y in meters from SW corner)
GPS: Individual tree GPS coordinates recorded

Species Identification: Field identification with photographic voucher
Validation: Cross-referenced with iNaturalist observations where possible

--------------------------------------------------------------------------------
DATA FIELDS
--------------------------------------------------------------------------------

1. tree_id
   Description: Unique identifier for each tree
   Format: Species code + sequential number (e.g., HE001 = Hauya elegans #1)
   Notes: Permanent ID for re-measurement tracking

2. species_scientific
   Description: Scientific binomial name
   Format: Genus species
   Authority: Verified via iNaturalist community and botanical references
   Example: Hauya elegans

3. species_common
   Description: Local common name(s)
   Format: Text
   Notes: Regional vernacular; may include indigenous names where known
   Example: Guayabón, Aguacatillo

4. family
   Description: Taxonomic family
   Format: Text
   Example: Onagraceae, Lauraceae

5. dbh_cm
   Description: Diameter at breast height (1.3 m)
   Format: Decimal
   Units: Centimeters (cm)
   Precision: ±0.5 cm (diameter tape measurement)
   Method: Measured at 1.3 m height on uphill side if on slope

6. height_m
   Description: Total tree height
   Format: Decimal
   Units: Meters (m)
   Method: Visual estimation
   Precision: ±1 m (LIMITATION: no clinometer used)
   Notes: Estimated from ground to highest live foliage

7. health_status
   Description: Overall tree condition assessment
   Format: Text
   Values: Healthy, Moderate, Poor, Dead
   Criteria:
     - Healthy: Full crown, minimal herbivory, no major structural damage
     - Moderate: Partial defoliation, moderate herbivory, minor damage
     - Poor: Severe defoliation, heavy herbivory, major structural damage
     - Dead: Standing dead or recently fallen

8. epiphyte_cover_pct
   Description: Visual estimate of trunk surface covered by epiphytes
   Format: Integer
   Units: Percent (%)
   Range: 0-100
   Method: Visual estimation
   Notes: Includes bryophytes, orchids, ferns, bromeliads

9. plot_x_m
   Description: X-coordinate within plot (east-west)
   Format: Decimal
   Units: Meters (m)
   Origin: Southwest corner (0,0)
   Range: 0-30

10. plot_y_m
    Description: Y-coordinate within plot (north-south)
    Format: Decimal
    Units: Meters (m)
    Origin: Southwest corner (0,0)
    Range: 0-30

11. gps_lat
    Description: GPS latitude
    Format: Decimal degrees
    Datum: WGS84
    Precision: ±5 m (consumer GPS)

12. gps_lon
    Description: GPS longitude
    Format: Decimal degrees
    Datum: WGS84
    Precision: ±5 m (consumer GPS)

13. notes
    Description: Qualitative observations
    Format: Text
    Examples: "Buttress roots", "IUCN Endangered", "Heavy orchid load"

LIMITATIONS
--------------------------------------------------------------------------------
1. Baseline census only (no temporal replication yet)
2. Height estimated visually (no clinometer)
3. Epiphyte cover visual estimate (no quantitative biomass measurement)
4. GPS precision ±5 m (consumer-grade device)
5. Small sample size (single 900 m² plot)
6. No seedling or sapling (<10 cm DBH) census

--------------------------------------------------------------------------------
DATA QUALITY STATEMENT
--------------------------------------------------------------------------------
This baseline census provides a complete inventory of woody stems ≥10 cm DBH 
within the permanent plot. While height estimation and epiphyte cover are 
qualitative, DBH measurements are standardized and repeatable. The dataset 
is suitable for:
  - Density and basal area calculation
  - Species composition analysis
  - Baseline for long-term demographic monitoring

Formal dendrometry (clinometer heights, precise basal area) recommended for 
future censuses to improve quantitative rigor.
